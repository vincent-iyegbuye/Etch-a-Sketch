# Etch-a-Sketch (Sketchpad Project)

## Introduction
This project is a browser-based sketchpad inspired by the classic Etch-A-Sketch toy.  
Users can draw by hovering over a grid, creating a pixel-style drawing effect.

## Features
- Dynamic grid generated with JavaScript
- Hover-based drawing effect
- Adjustable grid size (up to 100x100)
- Clear and regenerate sketchpad functionality
- Responsive layout using Flexbox

## Skills Demonstrated
Through this project, I practiced and strengthened my understanding of:

- DOM manipulation (creating and modifying elements dynamically)
- Event handling (mouseover and hover interactions)
- JavaScript logic and functions
- Flexbox layout for structuring the interface
- User input handling (prompt and validation)
- Writing clean and structured code

## How to Use
1. Open the webpage in your browser
2. Hover over the grid to draw
3. Click the button to generate a new grid size
4. Enter a value (maximum: 100) to create a new sketchpad
